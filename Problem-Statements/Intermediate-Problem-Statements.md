## Overview

The Intermediate Level problems are designed for participants who already understand basic cloud deployment concepts.  
These problems focus on monitoring, automation, system design, and security, which are essential for building reliable and production-ready cloud applications.

---

## Problem 6: Application Monitoring using CloudWatch

### Problem Statement  
Students must implement monitoring for application logs and system performance.

### Brief Description  
The objective is to observe application behavior and system resource usage to identify performance issues or failures.  
This problem emphasizes the importance of monitoring and proactive system management in cloud environments.

---

## Problem 7: Full DevOps Pipeline (Code to Deployment)

### Problem Statement  
Students must implement a complete DevOps pipeline that automates the process from code commit to application deployment.

### Brief Description  
The pipeline should reduce manual intervention and ensure reliable delivery of application updates.  
This problem focuses on understanding the full lifecycle of modern DevOps workflows.

---

## Problem 8: Multi-Container Application using Docker Compose

### Problem Statement  
Students must deploy a multi-container application consisting of multiple services working together.

### Brief Description  
Each service should run independently while communicating correctly with other services.  
This problem introduces service orchestration concepts and multi-service application design.

---

## Problem 9: Infrastructure Setup Documentation (IaC Concept)

### Problem Statement  
Students must design cloud infrastructure and document it clearly using diagrams and structured explanations.

### Brief Description  
The focus is on planning and architectural thinking rather than only deployment.  
Clear documentation and logical explanation of infrastructure components are key expectations.

---

## Problem 10: Secure Cloud Application with IAM and HTTPS

### Problem Statement  
Students must deploy a cloud application with strong security considerations.

### Brief Description  
The problem emphasizes secure access control and protection of application communication.  
Participants should demonstrate awareness of cloud security principles and responsible deployment practices.
